## Здравствуйте! 👋
  
Меня зовут Яна Кушнер. Я ИТ-аналитик с экономическим бэкграундом. Моя главная задача - переводить хаотичные запросы бизнеса на технический язык, понятный команде разработчиков, и выстраивать эффективную архитектуру процессов без рутины и ручного труда. 

Знания в экономике помогают видеть реальную выгоду для бизнеса: как сократить расходы, избавиться от ручной работы и ускорить запуск продукта. 
Чтобы адаптировать запросы заказчиков для команды разработки, я освоила нотации BPMN/UML и проектирование REST API. Инструмент SQL позволяет мне самостоятельно работать с базами данных и автоматизировать рутину, а построение CJM, USM помогает мне управлять и структурировать требования.
Я выступаю надежным связующим звеном между стейкхолдерами и разработчиками, полностью контролируя качество продукта на всех этапах — от составления технического задания до приемочных испытаний. 

Стек: UML, анализ вариантов использования, User Story Mapping, BPMN 2.0, определение минимально жизнеспособного продукта, определение функциональных и нефункциональных требований, протоколы испытаний (ПМИ), техническое задание (ТЗ), приемо-сдаточные испытания (ПСИ), Figma (прототипирование, дизайн), Miro (User Story Mapping), draw.io (UML, BPMN), Excel, Google Docs (документирование требований и спецификаций). 

Нахожусь в активном поиске работы
